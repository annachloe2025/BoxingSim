※3 ガードの型（押し続けで維持／一定時間で自動解除）は要相談。被弾系（のけぞり・ダウン・立ち上がり）の保持フレームも今後設定。

状態カード（1状態ずつの詳細定義）

待機・構え

分類：中立 ／ 起点
説明入力待ち。すべての行動の起点となる状態。
この間できることすべての入力を受付（攻撃・前進・後退・ガード）。
保持フレーム入力があるまで任意。時間では自動遷移しない。
出ていく遷移
  • 攻撃入力 → 攻撃：始動
  • 前進入力 → 前進
  • 後退入力 → 後退
  • ガード入力 → ガード
  • 被弾 → 被弾：のけぞり

前進

分類：移動
説明間合いを詰める。前へ移動する。
この間できること原則ロック（移動キャンセルの可否は要相談）。
保持フレーム技データの 始動＋発生＋硬直（暫定：1／4／2）。
出ていく遷移
  • フレーム経過（移動完了） → 待機・構え
  • 被弾 → 被弾：のけぞり

後退

分類：移動
説明間合いを離す。後ろへ移動する。
この間できること原則ロック（移動キャンセルの可否は要相談）。
保持フレーム技データの 始動＋発生＋硬直（暫定：1／4／2）。
出ていく遷移
  • フレーム経過（移動完了） → 待機・構え
  • 被弾 → 被弾：のけぞり

攻撃：始動

分類：攻撃（startup）
説明パンチの予備動作。まだ当たり判定はない。ここで被弾するとカウンター。
この間できること原則ロック（行動不可）。
保持フレーム技の始動F（暫定：ジャブ3／ストレート6）。
出ていく遷移
  • 始動F経過 → 攻撃：発生
  • 被弾（カウンター） → 被弾：のけぞり

攻撃：発生

分類：攻撃（active）
説明当たり判定がアクティブ。相手が間合い内なら接触（ヒット）。
この間できること原則ロック。
保持フレーム技の発生F（暫定：ジャブ2／ストレート3）。
出ていく遷移
  • 発生F経過 → 攻撃：硬直
  • 被弾（相打ち） → 被弾：のけぞり

攻撃：硬直

分類：攻撃（recovery）
説明打ち終わりの隙。いちばん被弾しやすいタイミング。
この間できること原則ロック（硬直キャンセルの可否は要相談）。
保持フレーム技の硬直F（暫定：ジャブ6／ストレート12）。
出ていく遷移
  • 硬直F経過 → 待機・構え
  • 被弾 → 被弾：のけぞり

ガード

分類：防御
説明攻撃を防ぐ構え。ガード中は被弾せず防御する。
この間できること押し続け型／一定時間型は要相談。
保持フレーム型により異なる（要相談）。
出ていく遷移
  • ガード解除 → 待機・構え
  • 攻撃を防御（ガード成功） → ガード硬直

ガード硬直

分類：防御
説明ガードで攻撃を防いだ直後の硬直。
この間できることロック。
保持フレームガード硬直F（要設定）。
出ていく遷移
  • ガード硬直F経過 → ガード

被弾：のけぞり

分類：被弾 ／ 現フェーズは観察のみ（未実装可）
説明攻撃を受けて仰け反る。
この間できることロック。
保持フレームのけぞりF（要設定）。
出ていく遷移
  • のけぞりF経過 → 待機・構え
  • 規定ダメージ超過 → ダウン

ダウン

分類：被弾
説明規定ダメージを超えて倒れる。
この間できることロック。
保持フレームダウンカウント（要設定）。
出ていく遷移
  • ダウンカウント → 立ち上がり
  • テンカウント → 終了（KO）

立ち上がり

分類：被弾
説明ダウンから復帰する動作。
この間できることロック。
保持フレーム立ち上がりF（要設定）。
出ていく遷移
  • 立ち上がり完了 → 待機・構え

この3つ（mermaid図／遷移マトリクス／状態カード）は同じ内容を別の見せ方にしたものです。 図で全体をつかみ、表で「どの状態で何が起きると何になるか」を一覧し、カードで1状態ずつ詳しく定義していけます。
