頭/腹打ち分け（段階2）を入れたあと、「正常な駆け引き」として立ち上がるべき構造はこう。

1. 頭/腹2択 vs ガード2 → 守備は1/2でしか防げない
2. 2択回避の最適解は **バックステップ**（射程外に出れば外れる）
3. バックステップを無制限に使えるとコーナーに追い詰められる
4. コーナー = 不利（後退不能・攻撃側有利F）
5. ゆえに **バックステップは回数制限** が必要（ラウンド内 N 回 / クールダウン式 など）

**Why:**
2026-05-26 段階2スコープ確認時にユーザーが提示。Phase 2 完成直後に観測される最初の問題（守備有利時間切れ）の解決ルートとして既に頭の中に道筋がある。

**How to apply:**
- 段階2 (頭/腹) 完成後の verify.js 観測で「バックステップ多発・時間切れ多発」が出たら、これは想定通り。次の手は「コーナー(D1) + バックステップ回数制限」の組み合わせ。
- 段階2 単独で「バックステップ抑止」のために変な係数いじりはしない。コーナー機構と一緒に解くべき問題。
- スタミナの現行コスト（ステップバック −1.0/F）は既に「無限には使えない」抑制になっているが、ガス欠まで使い切れる以上はまだ不十分。コーナー機構が入って初めて完成する想定。
